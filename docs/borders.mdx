# Borders

---

<br />
<br />

#### Usage

```css
border-radius: var(--border-radius-small);
border-radius: var(--border-radius-med);
border-radius: var(--border-radius-lrg);
```

```js
import borders from '@deck/css/variables/agoda/borders.scss';

console.log(borders.borderRadiusSml);
console.log(borders.borderRadiusMed);
console.log(borders.borderRadiusLrg);
```
